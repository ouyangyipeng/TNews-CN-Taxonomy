# 架构示意图设计方案

## 设计原则
- 顶会风格（ACL/EMNLP/NeurIPS）：简洁、信息密度高、配色克制
- 减少文字标注，多用图形符号（箭头、色块、图标）表达数据流
- 统一配色方案：主色 #2563EB（蓝）、辅色 #DC2626（红）、中性 #6B7280（灰）
- 所有维度标注使用等宽字体，与正文区分

## 图 1：BiLSTM + Bahdanau Attention 分类器架构

**文件名**: `figures/bilstm_arch.png`

**布局**: 从左到右的数据流，纵向展开 Attention 计算细节

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  "北 京 天 气 多 云 转 晴"                               │
│   ↓   ↓   ↓   ↓   ↓   ↓   ↓   ↓                        │
│  ┌──────────────────────────────────┐                   │
│  │        Embedding (300d)          │                   │
│  └──────────────────────────────────┘                   │
│   ↓   ↓   ↓   ↓   ↓   ↓   ↓   ↓                        │
│  ┌──────────────────────────────────┐                   │
│  │   → LSTM Layer 1 →               │                   │
│  │   ← LSTM Layer 1 ←               │                   │
│  │   → LSTM Layer 2 →               │                   │
│  │   ← LSTM Layer 2 ←               │                   │
│  └──────────────────────────────────┘                   │
│   h₁  h₂  h₃  h₄  h₅  h₆  h₇  h₈                      │
│   ↓   ↓   ↓   ↓   ↓   ↓   ↓   ↓                        │
│  ┌──────────────────────────────────┐                   │
│  │     Bahdanau Attention           │                   │
│  │                                  │                   │
│  │  eᵢ = vᵀ·tanh(W·s + U·hᵢ)       │                   │
│  │  αᵢ = softmax(eᵢ)               │                   │
│  │  c = Σ αᵢ·hᵢ                    │                   │
│  │                                  │                   │
│  │  [热力图色条: α₁ α₂ ... α₈]      │                   │
│  └──────────────────────────────────┘                   │
│              ↓                    ↓                      │
│           context c          last hidden h_T             │
│              ↓                    ↓                      │
│           ┌────────────────────────┐                     │
│           │    Concat (1024d)      │                     │
│           │    FC → ReLU → Drop    │                     │
│           │    FC → Softmax        │                     │
│           └────────────────────────┘                     │
│                      ↓                                   │
│              news_travel (15 classes)                    │
└─────────────────────────────────────────────────────────┘
```

**关键视觉元素**:
- BiLSTM 用双向箭头表示前向/后向，两层用不同深浅的蓝色区分
- Attention 权重用热力图色条（黄→红）直观展示
- 维度标注用小号灰色字体放在模块右侧
- 分类结果用圆角矩形框出，15 个类别用小点阵表示

## 图 2：Transformer Encoder 分类器架构

**文件名**: `figures/transformer_arch.png`

**布局**: 经典的自底向上堆叠结构，突出 Encoder Layer 内部细节

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  Input Tokens                                           │
│   ↓                                                     │
│  ┌──────────────────────────────────┐                   │
│  │   Token Embedding (512d)         │                   │
│  │   + Positional Encoding          │                   │
│  │   (sin/cos 波形图标)              │                   │
│  └──────────────────────────────────┘                   │
│   ↓                                                     │
│  ┌──────────────────────────────────┐                   │
│  │  ┌────────────────────────────┐  │                   │
│  │  │  Multi-Head Self-Attention │  │                   │
│  │  │  ┌──┐┌──┐┌──┐┌──┐         │  │                   │
│  │  │  │H1││H2││H3││H4│ ...×8   │  │                   │
│  │  │  └──┘└──┘└──┘└──┘         │  │                   │
│  │  └────────────────────────────┘  │                   │
│  │         ⊕  Add & LayerNorm       │                   │
│  │  ┌────────────────────────────┐  │  ×4 layers        │
│  │  │  Feed-Forward Network      │  │                   │
│  │  │  512 → 2048 → 512          │  │                   │
│  │  └────────────────────────────┘  │                   │
│  │         ⊕  Add & LayerNorm       │                   │
│  └──────────────────────────────────┘                   │
│   ↓                                                     │
│  ┌──────────────────────────────────┐                   │
│  │   Global Average Pooling         │                   │
│  │   (mask-aware, 忽略 PAD)         │                   │
│  └──────────────────────────────────┘                   │
│   ↓                                                     │
│  ┌──────────────────────────────────┐                   │
│  │   FC (512→256) → ReLU → Drop    │                   │
│  │   FC (256→15) → Softmax         │                   │
│  └──────────────────────────────────┘                   │
│   ↓                                                     │
│  Output: 15 classes                                      │
└─────────────────────────────────────────────────────────┘
```

**关键视觉元素**:
- Multi-Head Attention 用 8 个小方块并排表示 8 个头，每个头内部画一个简化的 Q·K^T 矩阵乘图标
- Positional Encoding 用正弦波形的小图标表示
- Encoder Layer 用大虚线框包裹，右侧标注 "×4" 表示堆叠 4 层
- Add & Norm 用 ⊕ 符号 + 小标签表示残差连接
- Global Average Pooling 用一条横线 + 向下箭头表示

## 通用设计规范

| 元素 | 规范 |
|------|------|
| 模块框 | 圆角矩形，线宽 1.5pt，填充色 #F0F4FF |
| 数据流箭头 | 实线箭头，线宽 1pt，颜色 #374151 |
| 维度标注 | 9pt 等宽字体，颜色 #6B7280，放在模块右侧或下方 |
| 特殊操作符号 | ⊕ (Add), ⊗ (Multiply), σ (Activation) |
| 配色 | 主色 #2563EB, 辅色 #DC2626, 强调 #F59E0B, 中性 #6B7280 |
| 字体 | 中文：思源黑体 Regular；英文/数字：Inter 或 Helvetica |
| 分辨率 | 300 DPI，宽度不超过 14cm（适配双栏论文） |
| 格式 | PNG（报告用）+ PDF（矢量备份） |
